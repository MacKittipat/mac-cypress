# Cypress experiment project

### Requirement 
- Node.js
- Docker

### Backend 

http://localhost:3000

```
cd node-express-realworld-example-app
npm install
npm run mongo:start
npm run start 
```

### Frontend 

http://localhost:4200

```
cd angular-realworld-example-app
npm install
npm run start 
```

### Cypress 

```
npm install
npx cypress open
```
